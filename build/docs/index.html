Commands

23 commands for design fluency. One skill, six categories, from shaping a new feature to hardening it for production.

Create

Build something new, from a blank page to a working feature.

/impeccable craft

Shape the design, then build it, all in one flow.

/impeccable impeccable

The design intelligence behind every command.

/impeccable shape

Think before you build. Produce a design brief through discovery, not guesswork.

Evaluate

Review what you have. Score it, critique it, find what to fix.

/impeccable audit

Five-dimension technical quality check with P0 to P3 severity.

/impeccable critique

A design review with scoring, persona tests, and automated detection.

Refine

Improve one dimension at a time: type, layout, color, motion.

/impeccable animate

Purposeful motion that conveys state, not decoration.

/impeccable bolder

Push safe designs toward impact without sliding into chaos.

/impeccable colorize

Add strategic color to monochrome interfaces without going garish.

/impeccable delight

Small moments of personality that turn functional into memorable.

/impeccable layout

Fix layout, spacing, and visual rhythm.

/impeccable overdrive

Push an interface past conventional limits. Shaders, physics, 60fps, cinematic transitions.

/impeccable quieter

Tone down designs that are shouting without losing their intent.

/impeccable typeset

Fix typography that feels generic, inconsistent, or accidental.

Simplify

Strip complexity. Remove what does not earn its place.

/impeccable adapt

Make designs work across screens, devices, and contexts without amputating features.

/impeccable clarify

Rewrite confusing UX copy so interfaces explain themselves.

/impeccable distill

Ruthless subtraction. Strip designs to their essence.

Harden

Make it production-ready. Edge cases, performance, polish.

/impeccable harden

Make interfaces production-ready. Edge cases, i18n, error states, overflow.

/impeccable onboard

Design first-run experiences, empty states, and paths to value.

/impeccable optimize

Diagnose and fix UI performance from LCP to bundle size.

/impeccable polish

The meticulous final pass between good and great.

System

Setup and tooling. Design system work, extraction, organization.

/impeccable document

Generate a spec-compliant DESIGN.md that captures your visual system so every AI agent stays on-brand.

/impeccable extract

Pull reusable components, tokens, and patterns into the design system.

/impeccable live

Iterate on UI in the browser. Pick an element, drop a comment, get three variants. Accept one and it writes to source.

/impeccable teach

Teach Impeccable who your product is for, once per project.
